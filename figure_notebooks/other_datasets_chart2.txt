**Notes regarding new column ("Contains Multi-panel Plots (Sub-figures)?"):** 
*   **"Yes"** is indicated if the source explicitly references the dataset containing multiple views/compound figures (e.g., MV Dataset) or providing evaluation tasks requiring multi-chart/multi-image cross-referencing setups (e.g., MultiChartQA-R, SlideVQA, SPIQA).
*   **"No"** is used where sources state an exhaustive list of visualization types (e.g. FigureQA only has line, dot-line, bar, pie) clearly excluding multi-panel setups, or when the dataset specifically dismantled compound figures during processing (e.g., Viziometrics). 
*   **"Partially"** denotes instances like SciCap where sub-figures were identified during preprocessing but the format of the final instances is not explicitly mapped.
*   **"Not specified"** is used when the sources broadly describe the contents (e.g., "various chart types") without mentioning panels, sub-figures, or multi-view plots.
